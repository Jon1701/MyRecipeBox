# RecipeBoxV2

## Installation

1. Clone repository

  ```
  git clone https://github.com/Jon1701/RecipeBoxV2.git
  ```

2. Install dependencies

  ```
  yarn install
  ```

3. Run server

  ```
  yarn run server
  ```

4. Run client

  ```
  yarn run client
  ```

5. View in browser: http://localhost:8080
